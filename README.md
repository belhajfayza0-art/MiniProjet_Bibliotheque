### Description

L'application **Gestion de Bibliothèque** est un logiciel développé en JavaFX 25 permettant de gérer les livres et les emprunts d'une bibliothèque. Elle offre une interface graphique intuitive avec des fonctionnalités complètes de gestion (CRUD), de recherche, de filtrage, de statistiques et d'export CSV.

### Prérequis

Avant de lancer l'application, assurez-vous d'avoir installé :

1. **Java JDK 25**
   - [Télécharger](https://adoptium.net/)

2. **JavaFX SDK 25.0.2**
   - [Télécharger](https://gluonhq.com/products/javafx/)

3. **IntelliJ IDEA**
   - [Télécharger](https://www.jetbrains.com/idea/)

4. **WAMP Server** (ou XAMPP/MAMP)
   - [Télécharger](https://www.wampserver.com/)

5. **MySQL** (inclus dans WAMP)

6. **Driver MySQL Connector/J**
   - [Télécharger](https://dev.mysql.com/downloads/connector/j/)
